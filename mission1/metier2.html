Developpeur Fullstack

Activités principale:

Le développeur full-stack est un développeur qui travaille autant sur la partie front-end et/ou mobile que back-end. Il peut gérer un projet dans son intégralité. Le full-stack maîtrise tout son environnement. Il est en relation directe avec le designer et le chef de projet

  • il travaille avec le chef de projet pour bien comprendre le besoin et découper les tâches ;
  • Le developpeur full-stack intègre au site le design ;
